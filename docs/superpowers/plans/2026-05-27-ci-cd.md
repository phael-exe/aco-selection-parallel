# CI/CD Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Configurar GitHub Actions CI com 5 jobs paralelos que compilam as 3 versões do ACO, verificam equivalência seq vs OpenMP em datasets pequenos, e fazem lint dos scripts Python, rodando em ≤4 min em cada PR para `main`.

**Architecture:** Um único workflow `.github/workflows/ci.yml` com jobs independentes (`build-sequential`, `build-openmp`, `build-cuda`, `lint`) rodando em paralelo, mais um job `equivalence` que depende dos dois primeiros e compara os CSVs de saída. Um script Python `scripts/compare_outputs.py` faz a comparação com tolerância de ponto flutuante.

**Tech Stack:** GitHub Actions, ubuntu-latest, g++, OpenMP (via libgomp já incluso no g++), nvcc via `Jimver/cuda-toolkit`, Python 3 + numpy, pytest, flake8.

---

## Contrato de Interface do Makefile

O CI assume as seguintes interfaces. O responsável por T01 (Criar Makefile) deve respeitá-las:

| Target | Efeito |
|--------|--------|
| `make sequential` | Produz binário em `./build/sequential` |
| `make openmp` | Produz binário em `./build/openmp` |
| `make cuda CI=1` | Compila fontes CUDA com `nvcc --compile` — sem produzir executável |

**CLI dos binários:**
```
./build/<version> <dataset_path> --seed <int> --output <output_path>
```
Exemplo: `./build/sequential data/baseline/haberman.csv --seed 42 --output results/ci/seq_haberman.csv`

```python
# scripts/tests/test_compare_outputs.py
import os
import sys
import numpy as np
import pytest

sys.path.insert(0, os.path.join(os.path.dirname(__file__), ".."))
from compare_outputs import compare


def _write(path, data, delimiter=";"):
    np.savetxt(path, data, delimiter=delimiter, fmt="%g")


def test_identical_files_pass(tmp_path):
    a, b = tmp_path / "a.csv", tmp_path / "b.csv"
    data = np.array([[1, 0, 1], [0, 1, 0]])
    _write(a, data)
    _write(b, data)
    assert compare(str(a), str(b)) is True


def test_different_values_fail(tmp_path):
    a, b = tmp_path / "a.csv", tmp_path / "b.csv"
    _write(a, np.array([[1, 0, 1]]))
    _write(b, np.array([[0, 1, 0]]))
    assert compare(str(a), str(b)) is False


def test_shape_mismatch_fail(tmp_path):
    a, b = tmp_path / "a.csv", tmp_path / "b.csv"
    _write(a, np.array([[1, 0]]))
    _write(b, np.array([[1, 0, 1]]))
    assert compare(str(a), str(b)) is False


def test_within_tolerance_pass(tmp_path):
    a, b = tmp_path / "a.csv", tmp_path / "b.csv"
    data = np.array([[1.0, 0.0, 1.0]])
    _write(a, data)
    _write(b, data + 1e-9)  # dentro de atol=1e-6
    assert compare(str(a), str(b)) is True


def test_outside_tolerance_fail(tmp_path):
    a, b = tmp_path / "a.csv", tmp_path / "b.csv"
    data = np.array([[1.0, 0.0]])
    _write(a, data)
    _write(b, data + 1e-4)  # fora de atol=1e-6
    assert compare(str(a), str(b)) is False


def test_comma_delimiter(tmp_path):
    a, b = tmp_path / "a.csv", tmp_path / "b.csv"
    data = np.array([[1, 0, 1]])
    _write(a, data, delimiter=",")
    _write(b, data, delimiter=",")
    assert compare(str(a), str(b)) is True
```

```python
#!/usr/bin/env python3
import sys
import numpy as np


def load_csv(path):
    try:
        return np.loadtxt(path, delimiter=";")
    except ValueError:
        return np.loadtxt(path, delimiter=",")


def compare(file_a, file_b, atol=1e-6):
    a = load_csv(file_a)
    b = load_csv(file_b)

    if a.shape != b.shape:
        print(f"FAIL: shape mismatch {a.shape} vs {b.shape}", file=sys.stderr)
        return False

    if not np.allclose(a, b, atol=atol):
        diff = np.argwhere(~np.isclose(a, b, atol=atol))
        print(f"FAIL: {len(diff)} valores divergentes", file=sys.stderr)
        for idx in diff[:10]:
            key = tuple(idx)
            print(f"  {key}: {a[key]:.8f} vs {b[key]:.8f}", file=sys.stderr)
        return False

    print(f"OK: outputs equivalentes ({a.shape})")
    return True


if __name__ == "__main__":
    if len(sys.argv) != 3:
        print(f"Uso: {sys.argv[0]} <arquivo_a> <arquivo_b>", file=sys.stderr)
        sys.exit(2)

    ok = compare(sys.argv[1], sys.argv[2])
    sys.exit(0 if ok else 1)
```

### Task 2: Workflow GitHub Actions

**Files:**
- Create: `.github/workflows/ci.yml`

- [ ] **Step 1: Criar o diretório e o workflow completo**

```bash
mkdir -p .github/workflows
```

Criar `.github/workflows/ci.yml` com o conteúdo abaixo:

```yaml
name: CI

on:
  pull_request:
    branches: [main]
  push:
    branches: [main]

jobs:
  build-sequential:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4

      - name: Build sequential
        run: make sequential

      - name: Run on haberman
        run: |
          mkdir -p results/ci
          ./build/sequential data/baseline/haberman.csv --seed 42 --output results/ci/seq_haberman.csv

      - name: Run on heart_failure
        run: |
          ./build/sequential data/baseline/heart_failure.csv --seed 42 --output results/ci/seq_heart.csv

      - name: Upload artifacts
        uses: actions/upload-artifact@v4
        with:
          name: seq-outputs
          path: results/ci/seq_*.csv

  build-openmp:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4

      - name: Build OpenMP
        run: make openmp

      - name: Run on haberman
        run: |
          mkdir -p results/ci
          ./build/openmp data/baseline/haberman.csv --seed 42 --output results/ci/omp_haberman.csv

      - name: Run on heart_failure
        run: |
          ./build/openmp data/baseline/heart_failure.csv --seed 42 --output results/ci/omp_heart.csv

      - name: Upload artifacts
        uses: actions/upload-artifact@v4
        with:
          name: omp-outputs
          path: results/ci/omp_*.csv

  build-cuda:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4

      - name: Install nvcc
        uses: Jimver/cuda-toolkit@v0.2.14
        with:
          cuda: '12.4.0'
          method: 'network'
          sub-packages: '["nvcc"]'

      - name: Build CUDA (compile-only)
        run: make cuda CI=1

  lint:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4

      - name: Install flake8
        run: pip install flake8

      - name: Lint scripts/
        run: flake8 scripts/ --max-line-length=120 --select=E9,F

  equivalence:
    runs-on: ubuntu-latest
    needs: [build-sequential, build-openmp]
    steps:
      - uses: actions/checkout@v4

      - name: Download sequential outputs
        uses: actions/download-artifact@v4
        with:
          name: seq-outputs
          path: results/ci

      - name: Download OpenMP outputs
        uses: actions/download-artifact@v4
        with:
          name: omp-outputs
          path: results/ci

      - name: Install numpy
        run: pip install numpy

      - name: Compare haberman outputs
        run: python3 scripts/compare_outputs.py results/ci/seq_haberman.csv results/ci/omp_haberman.csv

      - name: Compare heart_failure outputs
        run: python3 scripts/compare_outputs.py results/ci/seq_heart.csv results/ci/omp_heart.csv
```
